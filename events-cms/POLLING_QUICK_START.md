# Polling Admin Panel - Quick Start Guide

## 🚀 Kaise Use Karein (How to Use)

### Admin Panel Main Access Kaise Karein

1. **Admin Panel Login Karein**
   - Browser main admin URL open karein
   - Email aur password se login karein

2. **Polling Section Main Jayen**
   - Left sidebar main "Module" section dekhein
   - "Polling" option par click karein (Blue "NEW" badge ke sath)
   - Ya direct URL: `your-domain.com/polls`

---

## 📝 Poll Kaise Banayein (Create Poll)

### Step-by-Step Process:

1. **"Create New Poll" Button Click Karein**
   - Polls list page ke top-right corner main button hai

2. **Event Select Karein** (Required ⚠️)
   - Dropdown se event choose karein
   - Ye mandatory field hai

3. **Speaker Select Karein** (Optional)
   - Agar specific speaker ke liye poll hai to select karein
   - Nahi to "No Speaker" chhod dein

4. **Question Likhein** (Required ⚠️)
   - Poll ka main question type karein
   - Example: "Which session did you enjoy the most?"

5. **Timer Set Karein** (Required ⚠️)
   - Seconds main time set karein (10-300)
   - Recommended: 30-60 seconds
   - Default: 30 seconds

6. **Options Add Karein** (Minimum 2 Required ⚠️)
   - Har option ka text likhein
   - "Add Option" button se naye options add karein
   - Maximum 6 options allowed
   - Trash icon se option delete kar sakte hain

7. **"Create Poll" Button Click Karein**
   - Poll save ho jayega
   - Success message dikhai dega
   - Polls list page par redirect hoga

---

## ✏️ Poll Kaise Edit Karein (Edit Poll)

1. Polls list main **Edit (pencil) icon** click karein
2. Jo bhi changes chahiye wo karein
3. **"Update Poll"** button click karein
4. Done! Poll update ho jayega

---

## 👁️ Poll Results Kaise Dekhein (View Results)

### Option 1: Single Poll Details
1. Polls list main **View (eye) icon** click karein
2. Dekh sakte hain:
   - Complete question aur options
   - Kitne votes aaye har option par (progress bars)
   - Kon-kon ne vote kiya (voters table)
   - Event aur speaker information

### Option 2: Complete Analytics
1. Poll results page par jayen
2. Event select karein dropdown se
3. Poll choose karein
4. Speaker filter laga sakte hain (optional)
5. Complete analytics dekhein:
   - Total polls, votes, voters
   - Vote distribution
   - Detailed voter list

---

## 🔴 Poll Ko Live Kaise Karein (Make Poll Live)

### Method 1: List View Se
- Polls list main toggle switch use karein
- On = Live, Off = Offline

### Method 2: Detail View Se
- Poll detail page kholen
- **"Go Live"** button click karein
- Live hone par button green "Live" dikhega

⚠️ **Important**: Sirf live polls hi users ko dikhengi!

---

## 🗑️ Poll Kaise Delete Karein (Delete Poll)

1. Polls list main **Delete (trash) icon** click karein
2. Confirmation modal aayega
3. **"Delete Poll"** confirm karein
4. Poll permanently delete ho jayega

⚠️ **Warning**: Is action ko undo nahi kar sakte!

---

## 🔍 Polls Kaise Filter Karein (Filter Polls)

### Available Filters:

1. **Event Filter**
   - "Filter by Event" dropdown use karein
   - Specific event ke polls dekhne ke liye

2. **Speaker Filter**
   - "Filter by Speaker" dropdown use karein
   - Specific speaker ke polls dekhne ke liye

3. **Search**
   - Search box main question ka text type karein
   - Real-time search results

4. **Apply Filters**
   - "Apply" button click karein filters lagane ke liye
   - "X" icon se filters clear kar sakte hain

## ⚙️ Settings aur Validations

### Poll Requirements:
- ✅ Event: Required (mandatory)
- ✅ Question: Required (khali nahi ho sakta)
- ✅ Timer: Minimum 10 seconds
- ✅ Options: Minimum 2, Maximum 6
- ⚠️ Speaker: Optional (zaruri nahi)

### Timer Recommendations:
- **Quick Polls**: 10-20 seconds
- **Standard Polls**: 30-60 seconds (recommended)
- **Complex Polls**: 60-120 seconds

## 🎯 Best Practices

### 1. Question Likhne Ke Tips:
- Clear aur specific question likhein
- Simple language use karein
- 200 characters se kam rakhein

### 2. Options Banane Ke Tips:
- Concise option text (50 characters tak)
- 2-4 options best rehte hain UX ke liye
- 6 se zyada mat banayein

### 3. Live Management:
- Jab voting start karni ho tabhi live karein
- Voting period khatam hone par disable karein
- Results regularly monitor karein

## 🐛 Common Issues aur Solutions

### Problem 1: Poll Dikh Nahi Raha
**Solution**: 
- Check karein ki poll "Live" hai
- Event association sahi hai ya nahi verify karein
- Timer expire to nahi hua

### Problem 2: Results Update Nahi Ho Rahe
**Solution**:
- Page refresh karein
- Internet connection check karein
- Kuch seconds wait karke dobara try karein

### Problem 3: Delete Nahi Ho Raha
**Solution**:
- Live status disable karein pehle
- Admin permissions check karein
- Active voting sessions band karein

### Problem 4: Filters Kaam Nahi Kar Rahe
**Solution**:
- Clear filters button click karein
- Filters dobara se apply karein
- Page refresh karein
